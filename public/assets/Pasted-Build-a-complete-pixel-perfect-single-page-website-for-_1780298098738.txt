## VISUAL DESIGN SYSTEM

**Color Palette (replace Auxility's green with TimeGlobalTech's signature blue-green):**
- Primary background: #ffffff (white)
- Accent / CTA color: #00C48C (vibrant teal-green — used for buttons, highlights, glows)
- Secondary accent: #1A1A2E (deep dark navy — used for dark sections)
- Text primary: #0D0D0D
- Text secondary: #555555
- Tag/label color: #00C48C with low-opacity background

**Typography:**
- Display headings: "Syne" (Google Font) — bold, large, uppercase-styled
- Body text: "DM Sans" (Google Font)
- Code-style tags like `<why timeglobaltech>` and `<services>` rendered in monospace font (Fira Code or similar) in accent color

**3D Decorative Shapes:**
- Use CSS + SVG or Spline embeds to create floating abstract 3D shapes throughout the page (clusters of cubes, spirals, half-rounds, diamonds, rings, prisms) — rendered in teal/green gradient tones, similar to Auxility's lime-green glassmorphism shapes
- These shapes appear in: Hero, About, Services, Cases, and CTA sections
- All shapes have a slow infinite float/rotate CSS animation

**Animations:**
- All sections fade-in + slide-up on scroll (Intersection Observer)
- Navbar blurs/frosts on scroll
- CTA button has a gradient shimmer hover animation
- Infinite horizontal marquee for client logos
- Horizontal auto-scroll carousel for testimonials (pause on hover)
- Animated number counters (count up when scrolled into view)
